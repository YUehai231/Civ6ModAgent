# Texture System Mods

### Creating the Texture and adding it to the XLP

1. - Open `Asset Editor`.

- File -> New -> XLP.

<img src="Images/TextureSystem_1.png" alt="" />

1. - Change the **XLP Class** field to the desired class.

<img src="Images/TextureSystem_2.png" alt="" />

1. - Choose a package name for the XLP.

- Save the XLP.

- Under the **Entries** section, click on the *Add New* button.

- Click *Add Source File...* and select the image that should be used.

- Set the name of the Texture that will be created, or leave it as the default value.

- Press the *Import* button.

- Save the XLP now that it has changed.
