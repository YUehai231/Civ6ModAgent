# ColorKey Mods

### Creating the Texture and adding it to the XLP

Note about ColorKey Source Files:  Our ColorKey textures are 3D textures, with a resolution of 64 x 64 x 64.  This is typically represented by a 2048 x 128 texture band.
We support other resolutions, but this requires modifying the Texture Export Settings for the ColorKey.

See: [Texture System Mods](#Modding%5CTexture%20System%20Mods).  Replace %XLP_CLASS% with `ColorKey`.
See: [Add and Update Libraries in Mod Art File](#Modding%5CAdd%20and%20Update%20Libraries%20in%20Mod%20Art%20File).  Replace %LIBRARY_NAME% with `ColorKey`
